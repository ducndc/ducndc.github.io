## Scanning Network

![H1](/assets/img/networking/scanning.png)

### Host Discovery Techniques
Used to indentify the active/live system in the network

![H1](/assets/img/networking/discoveryPing.png)

To detect OS of the target machine

nmap -sV --script smb-os-discovery.nse <target>

![H1](/assets/img/networking/os_discovery.png)

### Port Scaning 

Categorized according to the type of protocol used for communication

![H1](/assets/img/networking/portScanning.png)

### Nmap Script Engine

smb-os-discovery is an inbuilt script that can be used for collecting OS information on the target machine through the SMB protocol

nmap -sV -p445 --script vuln <target>

![H1](/assets/img/networking/ms17_010.png)

### Example about the detail command for scaning

nmap -sV -O -sC -v -oN nmap-win7.txt -p- <target machine>
